# StudyPal

StudyPal is an AI-powered study assistant that turns your notes into summaries, flashcards and quizzes. Built for university and secondary school students.

### Features of StudyPal includes:

- Uploading of study materials

- AI generated summaries

- Practice quizzes and flashcards

- Flexible pricing plans

### With StudyPal, students no longer need to worry about reading long boring notes that feels discouraging.
